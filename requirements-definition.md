# クイズアプリの要件
- クイズはランダムで出題される
- プレイヤーはアカウントを作らずにクイズに挑戦できる
- クイズの形式は4択問題
- クイズの問題は複数選択形式で、すべての正解を選択した場合に正解となる
- 管理者は専用のログイン画面で認証を行って、管理画面にアクセスできる
- 管理者はクイズの追加、編集、削除ができる
- 管理者は、問題の解説を追加できる
- 問題を解くと、正解・不正解と、解説が表示される
- 管理者はクイズのカテゴリーを登録できる
- プレイヤーはクイズのカテゴリを選んで、クイズに挑戦できる
- クイズはランダムで出題される
- カテゴリーの中のクイズをすべて回答したら、リザルトが表示されて、何問正解できたかわかる
